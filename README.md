# 🃏 Truco Argentino (Godot 4)

Juego de **Truco Argentino** para un jugador contra una IA, desarrollado en **Godot 4** con **GDScript**. Incluye la variante con **comodines**.

## 🎮 Características

- Partida 1 vs IA con la lógica del truco argentino (envido, truco, retruco, vale cuatro).
- Variante con comodines.
- Hecho 100% en Godot 4 + GDScript.

## ▶️ Cómo correrlo

1. Instalá [Godot 4](https://godotengine.org/download).
2. Cloná el repo y abrí `project.godot` desde el editor de Godot.
3. Ejecutá con **F5**.

## 🗂️ Estructura

```
├── scenes/        # Escenas del juego
├── scripts/       # Lógica en GDScript
├── assets/        # Recursos gráficos
├── Control.gd     # Controlador principal de UI
├── gestor.gd      # Gestor de la partida / reglas
└── project.godot  # Configuración del proyecto
```
